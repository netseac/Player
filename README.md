# player
Simple command-line golang mp3 player
Example usage

### Step 1

```
go get github.com/netseac/player
```

### Step 2

create main.go file
```
package main

import "github.com/netseac/player"

func main() {
	player.Run()
}
```

### Step 3

```
go run main.go -path=YOUR MP3 MUSIC FOLDER PATH
```

### No more steps, Enjoy!
